<br>
<p>Isometric pirate ship game where you dodge and destroy the royal navy that is pursuing you!</p>
<h3>Controls:</h3>
<ul><li>Accelerate (move forwrd) - W</li><li>Turn left/right - A or D</li><li>Shoot - Spacebar</li><li>Boost - Press shift when drift bar is full&nbsp;</li><li>Ability - 1-3</li></ul>
<p>Boost into enemies to ram them. Ramming them does damage.<br>Collect the purple orbs to level up. Levelling up grants you upgrades.</p>
<p>Made in 1 month&nbsp;for the <a href="https://itch.io/jam/game-off-2025" target="_blank">Github Game Off 2025</a></p>
